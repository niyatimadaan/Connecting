# Connecting
An app build for teachers to keep a track of the students and assign them with homework.

This app is built using flutter. It uses firebase database for storing information.

It contains:
<ul>
  <li>A bottom navigation bar</li>
  <li>Scrollable students list which can be added, edited and deleted</li>
  <li>Add student page</li>
  <li>Update student page</li>
  <li>Profile Page with Scrollable TimeTable</li>
</ul>
